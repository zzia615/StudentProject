# zxksglxtWeb221230
## 在线考试管理系统（webform+sqlserver）
### 开发语言：C#
### 数据库：SQL SERVER 2008 R2
### 主要功能：维护题库、维护试卷、维护用户、分配考生考试
### 用户角色：考生、管理员
